Данные карты

Цвет
Цвет фона карты.
Момент
Дата и время карты. Могут принимать значение now - тогда заменится на текущее время/дату
Место
Место карты. Широта и долгота могуть быть в произвольной форме, например 12N0.12' 13.349W 45.5E12.3"
Управление точками
Видно (<alt>+v) - точка отрисовывается в карте и участвует в аспектах
Аспектируется (<alt>+d) - точка принимает аспекты от других точек
Аспектирует (<alt>+g) - точка отбрасывает аспекты к другим точкам
Инфо (<alt>+i) - информация точки (координаты) выводятся в окно с картой

Возможно управление многими точками сразу, одновременно выделив несколько с нажатым <ctrl> или <shift>

Атлас
Данные на русском языке по 13тыс городам Мира. Временые зоны пока только у городов России. При поиске можно использовать метасимволы % (заменяет несколько любых символов) и _ (заменяет один любой символ), например Бо%ск (Бобруйск, Богушевск, Боровск...) _р_ (Арп, Ири, Крю...)
