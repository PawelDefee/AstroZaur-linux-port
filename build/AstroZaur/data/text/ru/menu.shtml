Программа
Загрузить - загрузить данные в программу.
Сохранить - сохранить все состояние программы в файл
Очистить - привести програму в первоначальный вид. Удаляются все данные
Выход - выйти из программы
Окно
Загрузить - загрузить данные в это окно. Создаются новые карты, загружаются аспекты, переключается активная карта
Создать - открыть новое окно
Сохранить картинку - вид окна сохраняятся в файл astrozaur.png
Карта
Создать - создать новую карту. После создания карты открывается диалог для ее редактирования
Выбор - выбрать активную карту для редактирования, вращения, аспектации итд. Первые 9 карт выбираются с помощью кнопок от 1 до 9
Аспекты - посмотреть аспекты от активной карты до выбранной. Первые 9 карт аспектируются с помощью сочетаний Ctlr+1 ... Ctrl+9
Крутить - изменение времени активной карты с определеным шагом. Выбранный шаг запоминается для каждой карты индивидуально.
Сохранить - сохранить данные активной карты - момент, место, видимость точек, коэфициенты дирекций/прогрессий...
Загрузить - прочитать данные в активную карту - момент, место, видимость точек, коэфициенты дирекций/прогрессий... Изменаются только те данные которые присутствуют в файле. Таким образом можно менять только местоположение карты или только видимость/аспектированность точек, а для дирекций/прогрессий - коэффициенты, итд итп
Править - редактирование данных карты и видимость/аспектация ее точек
Скрыть/Показать - скрыть если активная карта видима или показать если нет
Старт - выбрать точку активной карты как стартовую для отображения всех карт
